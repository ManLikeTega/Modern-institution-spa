## üöÄ Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language**: JavaScript
- **Package Manager**: npm

### Installation

Build Modern Institution SPA from source and install dependencies:

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/ManLikeTega/Modern-institution-spa
    ```

2.  **Navigate to the project directory:**

    ```bash
    cd Modern-institution-spa
    ```

3.  **Install the dependencies:**

    ```bash
    npm install
    ```

### Usage

Run the project with:

```bash
npm start
```

### Testing

Run the test suite with:

```bash
npm test
```
